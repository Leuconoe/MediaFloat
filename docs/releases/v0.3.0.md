# MediaFloat v0.3.0

MediaFloat v0.3.0 focuses on permission guidance, layout stability, and internal refactoring for better maintainability.

## Highlights

- **Permission guidance on launch**: When no permissions are granted, the app now automatically navigates to the Support screen where each required permission is listed with a clear explanation of why it is needed and a direct button to open the relevant settings page.
- **MIT License and privacy notice**: The Support screen now shows the MIT license and explicitly states that no personal information is collected, stored, or transmitted.
- **Layout boundary refactor**: Runtime and media responsibilities have been extracted into focused helpers (`MediaControllerSelector`, `MediaRecoveryPolicy`, `MediaArtworkCandidateResolver`, `OverlayLayoutCalculator`, `OverlayBitmapLoader`, `OverlayPresentationSpec`) to reduce coupling and improve testability.
- **Removed Product Constraints card**: The Support screen is now cleaner and more focused on what users actually need.
- **Auto-increment bundle version**: The release workflow now automatically bumps the `versionCode` during release builds to prevent Play Store conflicts.

## Notes

- Edge offset preset controls are temporarily disabled and will return in a future release.
- Thumbnail helper extraction is complete but the full thumbnail UI integration is still in progress.
- Existing permission and runtime requirements remain unchanged.
