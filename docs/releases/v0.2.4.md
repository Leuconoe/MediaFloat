# MediaFloat v0.2.4

MediaFloat v0.2.4 focuses on layout polish and thumbnail support for the floating widget.

## Highlights

- Normalized left drag-handle spacing to match the right side
- Added media thumbnail support with a fallback placeholder when artwork is unavailable
- Kept the title strip always visible to prevent layout flicker during track changes
- Moved sidebar placement and thumbnail toggle controls into Settings for easier access

## Notes

- Edge offset preset controls are temporarily disabled and will return in a future release
- Thumbnail fallback uses a styled placeholder box when no artwork is available from the media session
- Existing permission and runtime requirements remain unchanged
