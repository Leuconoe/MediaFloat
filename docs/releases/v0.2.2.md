# MediaFloat v0.2.2

MediaFloat v0.2.2 focuses on making the floating widget easier to read while media is playing.

## Highlights

- Added current-media title rendering to the floating widget overlay
- Kept the title strip width locked to the full widget width for Compact, Standard, and Large presets
- Added marquee-style scrolling for long titles without collapsing the title bar when the text is short
- Kept the runtime overlay and in-app preview aligned on the same title strip layout rules

## Notes

- Title metadata prefers `DISPLAY_TITLE` and falls back to `TITLE` when needed
- This release changes the overlay presentation only; it does not add thumbnail artwork or new transport controls
- Existing permission and runtime requirements remain unchanged

## Validation

- Passed `./gradlew.bat testDebugUnitTest`
- Passed `./gradlew.bat compileDebugAndroidTestKotlin`
- Passed `./gradlew.bat assembleDebug`
- Passed `./gradlew.bat connectedDebugAndroidTest`

## Feedback focus

The most useful feedback for `v0.2.2` is:

- title readability across different widget sizes and themes
- marquee smoothness with long track or video titles
- media-app compatibility for title updates when playback metadata changes quickly
