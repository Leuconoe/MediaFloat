# MediaFloat v0.2.1

MediaFloat v0.2.1 is a focused follow-up release.

## Highlights

- Added a launcher shortcut that stops the widget, paired with the existing launch shortcut
- Kept shortcut activation on the same runtime contract used by the app itself
- Renamed the package/application id to `sw2.io.mediafloat`
- Hardened runtime entry paths for notification settings, overlay start, and launcher shortcut flows
- Stabilized overlay drag updates so moving the widget no longer crashes the app
- Prepared refreshed screenshots and listing metadata for broader distribution
- Prepared signed Android App Bundle output for Play Store submission

## Notes

- This release does not change the core widget family or permission model
- Release and distribution assets now align with the verified app behavior and current UI surfaces
- Google Play should use the signed `.aab`, while APK assets remain useful for sideloading and other distribution channels
