# MediaFloat v0.2.3

MediaFloat v0.2.3 is a small follow-up release focused on stability and launcher convenience.

## Highlights

- Kept the floating title strip visible during temporary metadata gaps between tracks
- Added a `Toggle widget` launcher shortcut next to the existing launch and stop shortcuts
- Deferred thumbnail artwork support for a later release after real-device validation showed inconsistent behavior

## Notes

- The overlay still prefers current media title metadata and keeps the title surface stable even when the next item has not published text yet
- Thumbnail artwork is intentionally not part of this release
- Existing permissions and runtime entry requirements remain unchanged
