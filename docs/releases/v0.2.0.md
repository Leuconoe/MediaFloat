# MediaFloat v0.2.0

MediaFloat v0.2.0 is the first full product-shape release of the current single-module Android app.

This version completes the current cycle around app-language support, runtime recovery, a cleaner in-app information architecture, and release polish.

## Highlights

- App-language support for `System default`, English, Korean, Chinese, Japanese, Spanish, and French
- Main, Settings, Advanced, Support, and hidden Debug surfaces with simplified control grouping
- Real widget controls for size, width, opacity, sidebar side, and theme presets
- Media-session recovery improvements after temporary interruption paths
- Safer runtime entry handling for notification settings, overlay start, and launcher shortcut activation
- Apache-2.0 repository license and refreshed README, changelog, and release notes

## Setup notes

Language selection lives in `Advanced`. `System default` remains the first option and clears any app-specific override.

The overlay runtime still depends on Android overlay access, notification-listener access, and visible app notifications. If any of those pieces are missing, the app stays in a readable blocked state and exposes shortcuts back to the required system settings.

## Known limitations

- The overlay still supports a single horizontal control family
- Freeform resizing and freeform button placement are not supported
- Validation has been strong on emulator-based setup and runtime checks, but target-device testing is still recommended before wider use

## Feedback focus

The most useful feedback for `v0.2.0` is:

- translation quality across the shipped app-language set
- locale switching behavior on Android 13+ and Android 10-12 devices
- media-app compatibility and overlay recovery behavior across OEM builds
- drag reliability, sidebar placement, and overall widget readability across theme presets
